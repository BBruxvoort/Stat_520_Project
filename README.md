# STAT 520: Data Mining and Multivariate Statistics - Skills Showcase

## Project Overview
This project demonstrates the comprehensive statistical and data mining techniques learned in STAT 520. It serves as a portfolio piece showcasing proficiency in multivariate analysis, classification, clustering, and other advanced statistical methods.

## Project Objectives
Apply and demonstrate mastery of key data mining and multivariate statistical techniques covered in the course, including:
- Principal Component Analysis (PCA)
- Cluster Analysis
- Discriminant Analysis
- Classification methods
- Multivariate regression techniques
- Dimension reduction methods

## Methodology

### Techniques Demonstrated
- **Exploratory Data Analysis**: Multivariate visualization and summary statistics
- **Dimension Reduction**: PCA and factor analysis
- **Clustering**: K-means, hierarchical clustering, model-based clustering
- **Classification**: Various supervised learning approaches
- **Multivariate Modeling**: Multivariate regression and ANOVA

### Implementation
- Real-world dataset application
- Model building and selection
- Performance evaluation and interpretation
- Visualization of multivariate relationships
- Statistical validation
